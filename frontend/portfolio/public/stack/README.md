Local Tech Stack Icons

Drop your icon files here to override/replace tech logos used in project cards.

How it works
- We try `/stack/<name>.svg` first
- If missing, we fallback to `/stack/<name>.png`
- If still missing, we fallback to `/stack/_fallback.svg`

Tips
- Keep icons around 18–24px square for the current UI.
- Prefer SVG for sharp rendering; use PNG as a fallback.
- You can add more icons—just add the file and map its name in STACK_ICON_KEYS.
